TON Blockchain smart contracts event indexing service

up the service:
```
docker-compose -f <CONFIG> up --build -d <SERVICE>
```

stop the service: 
```
docker-compose -f <CONFIG> stop <SERVICE>
```

down all:
```
docker-compose -f <CONFIG> down
```
